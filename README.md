# SSTCN_Sea-Turtle-Conservation-Network_V
SSTCN (Students' Sea Turtle Conservation Network) is a volunteer organization dedicated to protecting endangered sea turtles through beach patrols, nest conservation, hatchling releases, awareness programs, and community participation. It promotes marine conservation and environmental sustainability.
# SSTCN Frontend

## Overview

The SSTCN Frontend is a responsive web application developed for the Students' Sea Turtle Conservation Network (SSTCN). It provides an intuitive and user-friendly interface to promote sea turtle conservation, allowing users to explore conservation initiatives, register as volunteers, view events, and learn about marine biodiversity.

## Features

* Responsive and modern user interface
* Home page with conservation awareness
* Volunteer registration and login pages
* Event and activity information
* Easy navigation with React Router
* Mobile-friendly design

## Technologies Used

* React.js
* HTML5
* CSS3
* JavaScript
* React Router

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/SSTCN-Frontend.git
```

2. Navigate to the project folder:

```bash
cd SSTCN-Frontend
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

## Folder Structure

```text
src/
├── assets/
├── components/
├── pages/
├── App.jsx
├── main.jsx

public/

```
